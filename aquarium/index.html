Aquarium growth sandbox

A structure grown inside the tank from an editable rulebook. Edit the rules, press Grow, and watch it accrete from its base. This is the Phase‑0 previewer described in PLAN.md.

Rulebook syntax

Parameters: key = value (angle, length, thickness, taper, fatten, iterations, jitter, gravity, leafsize, seed, axiom).

Rules rewrite a symbol each iteration: X -> F[+X][-X]. Weighted/random rules: X ->(0.3) F X.

Turtle symbols:

  • F grow a segment · f move, no growth
  • + - turn · & ^ pitch · \ / roll · | u-turn
  • [ ] branch push/pop · ! . thin/thicken
  • ' shift colour · L place a polyp/frond
